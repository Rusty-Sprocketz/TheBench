# v1.2.0 Release Notes
**Date:** 2026-02-12

## Changes
- Replaced "Done" tab badges with parsed verdict ratings (e.g. "Strong Hire 8.3")
- Added rich summary panel with overall rating, per-agent score bars, and summaries
- Softened Architect and Operator prompts for non-coding candidates (enterprise/AI direction focus)
- Enforced exactly 3 questions per interviewer (previously 3-4)
- Added markdown rendering in chat messages
- Added `parseVerdict` utility for extracting structured data from verdict blocks

- `api/architect.js` — softened prompts for non-coding candidates
