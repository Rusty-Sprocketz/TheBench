### Consolidated pipeline into single serverless function
- Vercel Hobby plan has a 12 serverless function limit; the 8 separate pipeline routes + cron pushed it over

### All buttons now tear down deployed apps
- Clear, Start Over, and Cancel Pipeline all call `api.cleanup()` to delete the demo project
- Previously only Start Over cleaned up; Clear just reset the UI and Cancel did nothing

### Fixed stale closure in cleanup callbacks
- `useCallback` captured stale `projectName` from render closure
- All cleanup actions now read from a `projectNameRef` ref that always holds the current value
